Удобный календарик (усовершенствованный) без использования ActiveX контролов

РРазделы:  Интерфейс

Описание: пример по использованию псевдо элемента MonthViewControl. Очень удобен для ввода даты и никаких проблем с совместимостью
Данная версия Датепикера содержит следующие обновления и исправления:

- При клике мышкой за пределами календаря - закрывается
- Если импортировать в проект форму выбора месяца то она будет доступна при клике на месяц
- Если дата в поле для ввода не указана то датепикер берет последнее
значение даты введенное через Датепикер.
- Последнее введенное значение сохраняется в реестре раздельно
для каждого компьютера и проекта
- После обновления значения даты в поле для ввода вызывается
событие "После обновления" этого поля.
- Добавлена возможность автоматического экспорта в свой проект
