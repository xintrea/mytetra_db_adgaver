Внешние Данные

  • Автоматическое подключение внешних таблиц, полученных при разбиении исходной базы данных на таблицы и объекты
  • Подключение всех таблиц указанного файла MDB (DAO)
  • Подключение таблиц MySQL без DSN (ADOX или DAO)
  • Подключение таблиц SQL Server без DSN (ADOX)
  • Подключение указанной таблицы определённого MDB файла (DAO)
  • Проверка качества подключения таблиц
  • Удаление всех подлинкованных таблиц
