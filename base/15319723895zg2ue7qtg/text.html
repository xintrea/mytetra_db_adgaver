Управление режимом видимости форм

Автор: Дмитрий Сонных  sonni-dim@mail.ru

Иногда бывает необходимо изменить режим видимости уже открытых форм. Ниже приведены подпрограммы, которые легко могут сделать это. Их лучше всего разместить в модуле. На основе их могут быть написаны подпрограммы, меняющие другие параметры открытых форм.

'-------------------------------------------------------------------

Public Sub IsFormVisibleOff(Name As String)

' подпрограмма, выключающая видимость формы

Dim FP As Form

For Each FP In Forms

If FP.Name = Name Then

FP.Visible = False

Exit For

End If

Next

End Sub

'-------------------------------------------------------------------

Public Sub IsFormVisibleOn(Name As String)

' подпрограмма, включающая видимость формы

Dim FP As Form

For Each FP In Forms

If FP.Name = Name Then

FP.Visible = True

Exit For

End If

Next

End Sub

'-------------------------------------------------------------------
