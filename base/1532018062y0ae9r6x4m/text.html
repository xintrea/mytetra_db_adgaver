Путь к Папке и Файлу текущего приложения и Путь к Access (MSACCESS.EXE)

Путь к папке текущего приложения:

sPath = CurrentProject.Path 'Полный путь к папке текущего приложения (без слеша на конце)



Имя Файла текущего приложения:

TempFileName = CurrentProject.Name 'Имя файла (текущего )запущенного приложения



Путь к Файлу текущего приложения:

CurrentDB.Name 'Полный Путь к аппликации



Путь к файлу в папке текущего приложения:

Private Function PathToINI() As String

'Возвращает путь к файлу INI (Application.ini) внутри папки тек. приложения

'--------------------------------------------------------------------

Const INI_FileName As String = "Application.ini"

'Путь к Папке аппликации

PathToINI = CurrentProject.Path & "\" & INI_FileName

End Function
