Комментарии

Объект Panel представляет собой элемент управления, содержащий другие элементы управления. Можно использовать Panel Чтобы сгруппировать коллекции элементов управления, таких как группы RadioButton элементов управления. Как и в других контейнерных элементов управления, таких как GroupBox управления Panel элемента управления Enabled свойству false, содержащийся в Panel будут также отключены.

Panel Элемент управления отображается по умолчанию без границ. Позволяет отобразить стандартную или трехмерную границу BorderStyle Свойства, чтобы выделить область панели среди других областей формы. Поскольку Panel элемент управления является производным от ScrollableControl можно использовать AutoScroll свойство, чтобы включить полосы прокрутки в Panel элемента управления. Когда AutoScroll свойству true, все элементы управления, расположенных в пределах Panel (но вне его видимой границы), можно прокручивать с помощью присутствующих полос прокрутки.

Panel Элемент управления не отображается заголовок. Если необходим элемент управления аналогичен Panel можно отобразить заголовок см. в разделе GroupBox элемента управления.

Примеры

В следующем примере кода создается Panel управления Label и TextBox для Panel. Panel Элемент управления отображается с трехмерной границей, чтобы отличить where Panel размещен элемент управления относительно других объектов в форме. В этом примере требуется, что метод, определенный в этом примере вызывается из существующей формы, а System.Drawing к исходному коду формы добавлено пространство имен.

C++

VB

Public Sub CreateMyPanel()

Dim panel1 As New Panel()

Dim textBox1 As New TextBox()

Dim label1 As New Label()


' Initialize the Panel control.

panel1.Location = New Point(56, 72)

panel1.Size = New Size(264, 152)

' Set the Borderstyle for the Panel to three-dimensional.

panel1.BorderStyle = System.Windows.Forms.BorderStyle.Fixed3D


' Initialize the Label and TextBox controls.

label1.Location = New Point(16, 16)

label1.Text = "label1"

label1.Size = New Size(104, 16)

textBox1.Location = New Point(16, 32)

textBox1.Text = ""

textBox1.Size = New Size(152, 20)


' Add the Panel control to the form.

Me.Controls.Add(panel1)

' Add the Label and TextBox controls to the Panel.

panel1.Controls.Add(label1)

panel1.Controls.Add(textBox1)

End Sub
