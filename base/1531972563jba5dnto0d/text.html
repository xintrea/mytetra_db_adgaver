Выход из подчиненной формы по клавише [TAB]

Предлагаемый кусок кода переводит фокус из подчиненной формы - в указанный элемент (поле) главной по нажатию [TAB] - т.е. так же как и для обычных полей.
Удобно когда подчиненная не редактируемая, а в главной есть поле задания шаблона поиска по подчиненной.

(Код помещается в модуль подчиненной формы + Свойство подчиненной формы KeyPreview = On)

Private Sub Form_KeyDown(KeyCode As Integer, Shift As Integer)

On Error GoTo KeyDownERR

If KeyCode = vbKeyTab Then

KeyCode = 0

'Перемеещение фокуса на нужный элемент главной

Me.Parent!txtToFind.SetFocus 'На поле поика "txtToFind"

End If

Exit Sub

KeyDownERR:

KeyCode = 0

End Sub
