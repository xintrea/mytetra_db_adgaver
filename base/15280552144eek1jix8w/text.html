• [HIDE] - Скрытый контент

    Скрывает контент от пользователей.

    Пример:
    [HIDE]Скрытие от гостей[/HIDE]
    [HIDE=5]Скрытие по числу постов[/HIDE]
    [HIDE='post=5']Скрытие по числу постов[/HIDE]
    [HIDE='like=5']Скрытие по числу симпатий[/HIDE]
    [HIDE='paid=5']Скрытие по числу оплаченных складчин[/HIDE]
    [HIDE='organized=5']Скрытие по числу организованных складчин[/HIDE]
    [HIDE='month=5']Скрытие по числу месяцев с момента регистрации[/HIDE]
    [HIDE='group=3,4,24,25,26']Скрытие для групп Админы(3), Модеры(4), Орги(24), Партнеры(25), ЧК(26)[/HIDE]
    [HIDE='user=Andrey,Alexey']Скрытие для пользователей[/HIDE]
    [HIDE='user=143,2333,4543']Скрытие для пользователей[/HIDE]
    [HIDE='post=5;organized=5;group=1,2,4;']Скрытие по комбинации параметров[/HIDE]
    Вывод:
    Скрытый контент.
    Скрытие от гостей
    Скрытый контент. Для просмотра необходимо: 5 сообщений.
    Скрытие по числу постов
    Скрытый контент. Для просмотра необходимо: 5 сообщений, 5 организованных складчин, иметь статус Гость или Складчик или Модератор.
