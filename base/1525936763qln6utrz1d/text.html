'===================================================================================================

' ##### Добавить слайды из файла PPTX

'

'---------------------------------------------------------------------------------------------------

Sub AddSlideInFile()

Dim strFile As String

''В этом примере вставляет слайды 3 по 6 из C:\Ppt\Sales.ppt после слайда двух активной презентации.

strFile = "<path_to_folder_or_file>.pptx"

ActivePresentation.Slides.InsertFromFile strFile, 2, 1, 1

End Sub

'===================================================================================================


Вставка слайдов из файла в презентации, в указанном расположении. Возвращает целое число , представляющее номер слайды, вставленные.

Синтаксис

выражение. InsertFromFile ( Имя файла, индекса, SlideStart, SlideEnd )

выражение Переменная, которая представляет собой объект- Слайды .

Параметры


Имя

Обязательный или необязательный

Тип данных

Описание

Имя файла

Обязательный

String

Имя файла, содержащего слайдов, который необходимо вставить.

Index

Обязательный

Длинный

Номер индекса объекта слайда для вставки нового слайда после указанной коллекцией Слайды .

SlideStart

Необязательный

Длинный

Номер индекса первого объекта слайда в коллекции слайдов в файле с FileName идентификаторами.

SlideEnd

Необязательный

Длинный

Номер индекса объекта последний слайд в коллекции слайдов в файле с FileName идентификаторами.

Возвращаемое значение

Целое число

Пример

В этом примере вставляет слайды 3 по 6 из C:\Ppt\Sales.ppt после слайда двух активной презентации.

ActivePresentation.Slides.InsertFromFile _

"c:\ppt\sales.ppt", 2, 3, 6
