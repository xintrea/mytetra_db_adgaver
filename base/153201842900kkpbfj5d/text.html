Сумма по товару

Function esTotal(curPrice As Currency, curQuantity As Currency) As Currency

'es 01.07.2011

'Возвращает сумму по товару

'--------------------------------------------------------------------

'Аргументы:

' curPrice = Цена товара

' curQuantity = Кол-во товара (возможно дробное)

'--------------------------------------------------------------------

On Error GoTo TotalErr

esTotal = CCur(Round(curPrice * curQuantity, 2))

Exit Function

TotalErr:

esTotal = 0: Err.Clear

End Function
