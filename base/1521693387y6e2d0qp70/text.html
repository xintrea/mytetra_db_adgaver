Функции Visual Basic 5.0/6.0

RGB  

Описание

Функция возвращает цветовой код RGB.

Синтаксис

RGB(Red, Green, Blue)

Параметры

Red - значение типа Integer в интервале от 0 до 255, указывающее красный цвет.
Green - значение типа Integer в интервале от 0 до 255, указывающее зеленый цвет.
Blue - значение типа Integer в интервале от 0 до 255, указывающее синий цвет.

Возвращаемые значения

Возвращает значение типа Long.

Примечание

Аргументы, имеющие значение больше 255, округляются до 255.

Пример использования

1Private Sub Form_Load()
2  'меняем цвет формы на белый
3  BackColor = RGB(255, 255, 255)
4End Sub
