Утилита состоит из двух файлов: системной службы и консоли для настройки параметров. Не требует установки и дополнительных компонентов для работы. Служба через заданный промежуток времени строит HTML-список файлов, хранящихся в определённой папке и её подпапках. В список выводится имя файла, его размер, дата последнего изменения. Утилита может вывести в этом списке MD5 и/или SHA-1 хеш-сумму файлов. Путь к файлу может быть преобразован в кликабельную ссылку для загрузки этого файла, например, с FTP. Список может создаваться автоматически (при установленной службе) или вручную (без установки службы).
