Курсор Мыши - Показать "Часики"

В обычной ситуации Access и сам часы показывает неплохо - но если уж использовать это - то главное не забыть возвращать нормальный курсор (особенно при возникновении ошибок)- ибо часами по кнопкам щелкать не удобно.... :)

On Error GoTo ErrorPoint

DoCmd.Hourglass True 'Показать часики

'....

'куча всяких громоздких инструкций

'....

DoCmd.Hourglass False 'Вернуть нормальный курсор

Exit Sub


ErrorPoint:

DoCmd.Hourglass False 'Вернуть нормальный курсор

MsgBox Err.Description
