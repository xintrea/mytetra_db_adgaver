Кнопка (Command Button) - Изменение надписи на кнопке при наезде на неё мыши

Прислал: Дмитрий Сонных sonni-dim@mail.ru


Изменение надписи на кнопке при наезде на неё мыши За основу данного примера взята разработка Александра Юраги. Мною были внесены только небольшие добавления. Помещается с разрешения автора. При помощи небольших ухищрений можно придать своей программе вполне профессиональный вид. Вот пример того как можно менять цвет (шрифт, размер, содержание и т.д.) надписи на кнопке. Для этого используется свойство элемента MouseMove. Подробности смотри в примере.
