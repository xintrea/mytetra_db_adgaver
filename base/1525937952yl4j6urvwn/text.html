Возвращает объект классов FileDialog , представляющий один экземпляр диалогового окна файла. Только для чтения.

Синтаксис

выражение. Классов FileDialog ( Тип )

выражение Переменная, которая содержит объект приложения .

Параметры

ИмяОбязательный или необязательныйТип данныхОписание
TypeОбязательныйMsoFileDialogTypeТип диалогового окна для возврата.

Возвращаемое значение

Классов FileDialog

Комментарии

Значение параметра типа может быть одной из этих констант MsoFileDialogType .

|| |:-----| |msoFileDialogFilePicker| |msoFileDialogFolderPicker| |msoFileDialogOpen| |msoFileDialogSaveAs|

Пример

В этом примере отображается диалоговое окно " Сохранить как ".

Sub ShowSaveAsDialog()


Dim dlgSaveAs As FileDialog


Set dlgSaveAs = Application.FileDialog( _

Type:=msoFileDialogSaveAs)


dlgSaveAs.Show


End Sub



В этом примере отображается диалоговое окно " Открыть " и позволяет пользователю выбрать несколько файлов, чтобы открыть.

Sub ShowFileDialog()


Dim dlgOpen As FileDialog


Set dlgOpen = Application.FileDialog( _

Type:=msoFileDialogOpen)


With dlgOpen

.AllowMultiSelect = True

.Show

End With


End Sub
