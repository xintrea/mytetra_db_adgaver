Как настроить git через прокси в windows

Вся настройка укладывается в пару строк:

1. Прописываем наш прокси сервер в кофигурацию Git:

git config –global http.proxy http://user:password@myproxy.ru:3128

2. Чтобы удостовериться, что настройка применилась, достаточно ввести следующую строку:

git config –get –global http.proxy

3. Если настроики применились, можно начинать пользоваться:

git pull http://github.com/ВашУрл master

потребуется ввести логин и пароль от того сайта, где находится ваш репозитарий.


Также существует альтернативный способ настройки. Можно прописать вручную всё что обсуждалось выше.

В глобальный файл конфигурации C:\Documents and Settings\username\.gitconfig или, если репозиторий уже есть, в файл .git\config конкретного репозитория.

Параметр в файле конфигурации действует сразу для http:// и https://.

[http]

proxy = http://login:password@192.168.1.1:3128
