Площадка хранения металла, с козловым краном, решение выполнено такое:

т. к. столбы освещения по периметру с удаленностью 100-150 м, от административного здания - поставили на каждом столбе освещения комплект:

1. IP камера 2 Mp 2,8-12 ("Орбита- 7012", тек. цена 3000 руб)
2. WiFi "Ubiquiti, Nanostation (loco) M2" - (тек.цена 3300 руб)
3. "БП-3У источник питания уличный Элис" - (тек.цена 1000 руб) можно использовать и "Резерв-12/3У (12В, 3А)"
4. Перфорированная пластина 100х600 мм, на ней смонтирована IP камера+блок питания+Точка доступа
5. Шпилька длинная с резьбой с двумя гайками, шпильку сгибаем в форме буквы U, обхватываем столб и надеваем перфорированную пластину.

Цена вопроса на одну камеру получилась: около 8000 руб.

На козловом кране, который по рельсам двигается так же стоит под кабиной пилота IP камера:
1. IP камера 2 Mp 2,8-12 ("Орбита- 7012", тек. цена 3000 руб)
2. WiFi "Ubiquiti, PicoStation M2-HP" - (тек.цена 5500 руб) (круговая антенна т. к. кран двигается)
3. "БП-3У источник питания уличный Элис" - (тек.цена 1000 руб) можно использовать и "Резерв-12/3У (12В, 3А)"

Камера вместе с караном катается по площадке, двигается по крану вместе с кабиной
пилота и живое видео транслируется в регистратор и диспетчеру

В административном здании у диспетчера:
-центральная уличная точка WiFi Ubiquiti Rocket M2 - 6000 руб
-антенна центральная для Rocket M2 - Ubiquiti AirMax Omni AMO-2G10 -12000 руб
-видеорегистратор 8 каналов IP 1080P "ОРБИТА VP-7916" - 3500 руб текущая цена.
-жесткий диск 2 ТБ Seagate Surrvilance SV35 ST2000VX00 - 5400 р. текущая цена.
(гарантия сигейта 3 года, специальная серия для круглосуточной работы именно для систем видеонаблюдения, 1 млн часов наработки)
- 8 портовый сетевой коммутатор D-link 2108 (в т. ч. для проброса VPN)

Итого вышло на 5 камер в пересчете на текущие цены без стоимости работ около 73000 руб.

Это бюджетный вариант, который работает круглосуточно уже второй год.

Мощность точек доступа на столбах выбрана минимальная, с IP камер транслируется 1080P, поток CBR, с битрейтом 4 мбита/сек.

Что удобно это все точки Ubiquiti управляются из одной программы "aircontrol-1.1.01", можно выбрать каналы WIFI 13,14,15 на которых как правило всегда свободно
В каждой точке Ubiquiti есть график скорости передаваемого/принимаемого потока между, по которым очень удобно наглядно видно идут ли провалы по радиоканалу или нет. Мы добились стабильного ровного графика потока 4000 Мб/сек, с каждого столба, без провалов и всплесков.
