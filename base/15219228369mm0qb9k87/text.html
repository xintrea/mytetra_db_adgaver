Комментарии

A ProgressBar управления визуально показывает ход выполнения длительных операций в одном из трех стилей:

  • Сегментированный блоками увеличивается слева направо.
  • Непрерывная полоса, заполняется слева направо.
  • Блок, который прокручивается в ProgressBar в виде бегущей строки.

Style Свойство определяет стиль ProgressBar которая отображается. Обратите внимание, что ProgressBar элемент управления может быть только ориентирована по горизонтали. Пример создания вертикально ориентированного ProgressBar, в разделе ProgressBarRenderer класса. ProgressBar Управления обычно используется, когда приложение выполняет задачи, такие как копирование файлов или печать документов. Пользователи приложения могут решить не отвечает, если нет визуальную подсказку. С помощью ProgressBar в вашем приложении можно оповестить пользователя о том, что приложение выполняет длительную задачу и, по-прежнему отвечает приложение.

Maximum И Minimum определяют диапазон значений для отображения хода выполнения задачи. Minimum Свойство обычно устанавливается в значение 0 и Maximum обычно присваивается значение, определяющее завершение задачи. Например, чтобы правильно Показать ход выполнения при копировании группы файлов Maximum свойству может быть присвоено общее число файлов для копирования.

Value Свойство представляет ход выполнения приложения по выполнению операции. Значение, отображаемое ProgressBar только аппроксимирует текущее значение Value Свойства. В зависимости от размера ProgressBar, Value свойство определяет момент отображения следующего блока или увеличьте размер полосы.

Существует несколько способов изменения значения, отображаемого в ProgressBar отличный изменение Value свойство напрямую. Можно использовать Step для указания определенного значения приращения Value Свойства, а затем вызовите метод PerformStep метод для увеличения значения. Для изменения значения приращения, можно использовать Increment метод и укажите значение, на которое будет изменяться Value свойство.


Примечание

ForeColor и BackColor изменения для ProgressBar не будут учитываться, если визуальные стили включены на Windows XP Home Edition, Windows XP Professional, Windows Server 2003 и выше.

В примере используется Minimum и Maximum Свойства, чтобы указать диапазон для ProgressBar эквивалентный номеру файлов для копирования. В коде также используется Step свойство с PerformStep метод для увеличения значения ProgressBar во время копирования файла. В этом примере требуется наличие ProgressBar элемент управления, созданный вызываемой pBar1 создаваемый в Form именем создается и используется метод CopyFile (который возвращает логическое значение, указывающее на успешное завершение операции копирования файла), выполняющий операцию копирования файла. Данный код также требует, что массив строк, содержащих файлы для копирования создается и передается CopyWithProgress метод, определенный в примере, и метод вызывается из другого метода или события в Form.

' Perform the increment on the ProgressBar.

pBar1.PerformStep()
