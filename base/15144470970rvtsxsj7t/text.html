FSO: метод BuildPath

BuildPath

Синтаксис:

BuildPath(<Path>,<Name>)

Назначение: добавляет к заданному пути новое имя. Если необходимо, вставляется “\”.

Параметры:

  • <Path> – строка, путь.
  • <Name> – строка, имя файла.

Пример:

Set FSO = CreateObject(“Scripting.FileSystemObject”)

Path = FSO.BuildPath(“C:\Program Files”, “New Folder”)

MsgBox Path

Path = FSO.BuildPath(“C:\Program Files\”, “New Folder”)

MsgBox Path

Метки: BuildPath, CreateObject("Scripting.FileSystemObject"), FileSystemObject, FSO
