Кнопка (Command Button) - Изменение курсора при наезде на кнопку

Прислал: Дмитрий Сонных sonni-dim@mail.ru

  
Изменение курсора "мыши" при наезде на кнопку. Стандартные кнопки Акцесс не позволяют изменить вид маркера мыши при наезде на кнопку. А хотелось бы. Использование сторонних ActiveX элементов ставит под вопрос лёгкую переносимость базы с одного компьютера на другой. Приходится кроме базы тащить все эти библиотеки, там их подключать, а тут возможны "нюансы" ... Поэтому мною выбрана библиотека Microsoft Forms 2.0 Object Library. Она входит в состав Офиса, и если у Вас стоит Акцесс, то эта библиотека точно есть. Примечание для продвинутых: Для хранения изображений можно использовать элемент Microsoft Image ListControl из библиотеки MSCOMCTL.OCX (версия 6), или тот же Image ListControl из библиотеки COMCTL32.OCX (версия 5)
