Private Sub Form_Resize()

 

        SW# = Me.ScaleWidth

        SH# = Me.ScaleHeight

 

        Zw# = FSW / SW#

        Zh# = FSH / SH#

 

        On Error Resume Next '!!!

 

        For i% = 1 To nC

            cntList(i%).Width = Ws(i%) / Zw#

            cntList(i%).Height = Hs(i%) / Zh#

            cntList(i%).Top = Ts(i%) / Zh#

           

        Next i%

 

        On Error GoTo 0 '!!!

 

End Sub
