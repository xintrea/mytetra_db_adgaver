Возвращаемое значение

Type: System.Int32

Возвращает Integer значение, представляющее RGB-значение цвета из набора компонентов красного, зеленого и синего цвета.

Исключения

Exception Condition
ArgumentException

Green, Blue, или Red выходит за пределы диапазона от 0 до 255, включительно.

Комментарии

Методы и свойства, принимающие цвета приложения предположении, что спецификации число, представляющее RGB значением цвета. RGB-значение цвета задает относительную интенсивность красного, зеленого и синего цвета для отображения.

Если значение любого из аргументов RGB больше 255, используется значение 255.

Примеры

В этом примере показано, как RGB функция используется для возвращения целое число представляет RGB значение цвета.

VB

Dim red, rgbValue As Integer

Dim i As Integer = 75

' Return the value for red.

red = RGB(255, 0, 0)

' Same as RGB(75, 139, 203).

rgbValue = RGB(i, 64 + i, 128 + i)
