Комментарии

Этот модуль поддерживает ключевые слова языка Visual Basic и элементы библиотеки времени выполнения, которые получить текущую дату и время, вычисляют дату, возвращают дату или время длительности процесса.

Примеры

В этом примере используется Today свойство для возврата текущей системной даты.

Dim thisDate As Date

thisDate = Today
