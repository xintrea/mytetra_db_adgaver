Вариант1: Открыть форму сразу после запуска файла с макросом

Код Visual Basic

Sub auto_open ()                    'Макрос для автоматического запуска нужного Вам макроса

    Application.Run ("OpenUserForm")

End Sub

 

Sub OpenUserForm ()             'Нужный Вам макрос

    UserForm1.Show                'Показать Вашу форму!!!

End Sub




Вариант2:  Открыть форму в процессе выполнения Вашего макроса
В Вашем макросе в месте, где нужно вызвать форму напишите

UserForm1.Show 'Показать Вашу форму!!!

затем, в месте, где Вашу форму нужно скрыть - напишите

UserForm1.Hide 'Скрыть Вашу форму!!!
