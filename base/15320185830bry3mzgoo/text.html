Использование функций MS Excel в MS Access

По материалам: http://www.tech-archive.net/Archive/Access/microsoft.public.access.modulesdaovba/2005-07/msg01389.html

Если свою функцию писать влом, то можно использовать функции MS Excel в MS Access
01. В референсах VBE добавляем ссылу на Microsoft Excel XX.X Object Library
02. Пишем приблизительно такую функцию:


Public Function Days360(StartDate As Date, EndDate As Date) As Integer

'--------------------------------------------------------------------------

'Внимание!

'Требуется ссылка на Microsoft Excel XX.X Object Library

'--------------------------------------------------------------------------

Days360 = Excel.WorksheetFunction.Days360(StartDate, EndDate)

End Function



Achtung!
Требуется ссылка на Microsoft Excel XX.X Object Library
