Создание совокупной строки из нескольких полей с разделителями

Прислал: Георгий Матющенко

Если собирать String c общей информацией (например, клиента) через запятую, то при отсутствии значения в поле получаются лишние запятые, типа:
Иванов И.И.,,,СПб,,99 лет
, дабы этого не происходило можно использовать IIf(isNull(...) - но получается слишком громоздкая запись. Следующий пример вставляет запятые только при наличии значения в поле.

StringAll = [ValueOfField01] & ("," + [ValueOfField02]) & ("," + [ValueOfField03]) '.... и т.д.
