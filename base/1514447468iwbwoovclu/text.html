FSO: метод GetExtensionName

Синтаксис:

GetExtensionName(<Path>)

Назначение: возвращает расширение последнего компонента в заданном пути.

Параметры:

  • <Path> – строка, путь.

Пример:

Set FSO = CreateObject("Scripting.FileSystemObject")

Name = FSO.GetExtensionName("Q:\aaa\bred\MySuperFile.txt")

MsgBox Name

Метки: CreateObject("Scripting.FileSystemObject"), FSO, FSO.GetExtensionName, GetExtensionName, SetFSO
